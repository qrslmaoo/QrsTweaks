# QrsTweaks

**QrsTweaks** is an all-in-one local Windows optimizer built with Python and PySide6. It runs entirely offline with no external APIs or telemetry.

## Current Features

- **Windows Optimizer** – System-level performance and debloat tools  
- **Game Optimizer** – Dedicated gaming performance tweaks  
- **Password Manager** – Secure local password storage  

All features are executed locally on your machine.
